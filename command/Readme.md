#### Example

```bash
kubectl create namespace NAME 
```

### Secret
#### Example
```bash
kubectl create secret docker-registry SECRET-NAME --docker-server=REGISTRY --docker-username=USERNAME --docker-password=PASSWORD --docker-email=EMAIL -n NAMESPACE
```
